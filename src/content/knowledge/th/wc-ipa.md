# 🌊 WC IPA (West Coast IPA)

<div class="article-lead">

WC IPA — West Coast IPA — คือสไตล์ที่ **ทำให้คราฟต์เบียร์อเมริกันเป็นที่รู้จักไปทั่วโลก** เกิดที่แคลิฟอร์เนียและแปซิฟิกนอร์ธเวสต์ โดดเด่นด้วย **ความใสแจ๋ว ความขมของฮอปที่ชัดเจน และการใช้ฮอปอเมริกันแบบไม่ประนีประนอม** — คิดถึงกลิ่นยางไม้ ซิตรัส และ resin กับจบท้ายที่แห้งสะอาด

ถ้า Hazy IPA คือความนุ่มและฉ่ำ WC IPA คือ **ความกรุบกรอบ ขมชัดเจน และภูมิใจในความขมของตัวเอง**

</div>

---

## 🏯 ประวัติโดยย่อ

West Coast IPA เกิดขึ้นในช่วงปี 1970–80 เมื่อผู้ผลิตเบียร์อเมริกันเริ่มทดลองกับฮอปสายพันธุ์อเมริกันที่มีกลิ่นหอมเข้มข้นซึ่งปลูกในแปซิฟิกนอร์ธเวสต์ ต่างจาก British IPA แบบดั้งเดิม พวกเขาต้องการ **ความขมมากขึ้น รสฮอปมากขึ้น และโปรไฟล์มอลต์ที่สะอาดกว่า**

ผู้บุกเบิกอย่าง **Sierra Nevada (Pale Ale)** , **Anchor Brewing (Liberty Ale)** และต่อมา **Stone Brewing** และ **Russian River** ได้กำหนดสไตล์นี้ จนถึงช่วงปี 1990–2000 West Coast IPA กลายเป็นสไตล์ IPA ที่มีอิทธิพลมากที่สุดในโลก

ชื่อ "West Coast" มาจาก **ฝั่งตะวันตกของสหรัฐอเมริกา** — โดยเฉพาะแคลิฟอร์เนีย ออริกอน และวอชิงตัน

## 👅 รสชาติเป็นอย่างไร?

<div class="taste-box">

### 🎨 ลักษณะ
- **ใสแจ๋ว** — กรองจนเงาวับ
- สีทองถึงอำพันอ่อน
- ฟองขาวแน่น อยู่ได้ดี

### 👃 กลิ่น
- **ยางไม้ (Pine)** — กลิ่นสนสด ๆ
- **ซิตรัส** — เกรปฟรุต เปลือกส้ม มะนาว
- **Resin** — กลิ่นฮอปเหนียว ๆ ดุ้น ๆ
- **มอลต์สะอาด** — ขนมปังหรือแครกเกอร์ ไม่หวานคาราเมล
- ไม่มีกลิ่นผลไม้จากการหมัก — สะอาด

### 👄 รสชาติ
- **ความขมชัดเจน** — 50–70+ IBU คมและสะอาด
- **รสฮอปสน ซิตรัส** — จุดเด่น
- **มอลต์สะอาด** — คอยหนุนไม่แย่งซีน
- **คาราเมลน้อยมาก** — ปล่อยให้ฮอปเด่น
- **จบท้ายแห้ง กรอบ** — ขมติดปลายลิ้นอย่างพอใจ

### ✨ ความรู้สึก
- **บอดี้ปานกลาง** — ไม่บางไม่หนัก
- **กรอบ สะอาด** — จากการหมักแห้งและน้ำตาลทรายเล็กน้อย
- คาร์บอนปานกลาง
- **จบท้ายแห้ง** — ทำให้ดื่มได้เรื่อย ๆ ทั้งที่ขม

</div>

## 🍶 West Coast vs Hazy IPA

| ด้าน | WC IPA (West Coast) | Hazy IPA (NE Style) |
|---|---|---|
| ลักษณะ | ใสแจ๋ว | ขุ่นทึบ |
| ความขม | สูง คม (50–70+ IBU) | ต่ำ นุ่ม (30–50 IBU) |
| กลิ่นฮอป | ยางไม้ สน เกรปฟรุต | ผลไม้เมืองร้อน juicy |
| มอลต์ | สะอาด ประคอง | นุ่ม โอ๊ตตี้ |
| บอดี้ | ปานกลาง กรอบ | ปานกลางถึงเต็ม ปุยนุ่ม |
| จบท้าย | แห้ง ขม | กึ่งแห้ง ผลไม้ |

## 🌲 ฮอปอเมริกัน

West Coast IPA ถูกกำหนดโดยสายพันธุ์ฮอปอเมริกัน:

| ฮอป | รสชาติ |
|---|---|
| **Cascade** | เกรปฟรุต ดอกไม้ — คลาสสิก |
| **Centennial** | มะนาว สน กลมกล่อม |
| **Chinook** | สน เครื่องเทศ จัดจ้าน |
| **Simcoe** | สน เอิร์ธโทน เสาวรส |
| **Citra** | เมืองร้อน ซิตรัส เข้มข้น |
| **Amarillo** | ส้ม ดอกไม้ หวาน |
| **Mosaic** | เบอร์รี่ เมืองร้อน ซับซ้อน |

---

## 🍽️ อาหารที่เข้ากัน

ความขมจัดของ WC IPA ตัดกับอาหารมัน เผ็ด และเข้มข้นได้ดี

| เมนู | ทำไมถึงเข้ากัน |
|---|---|
| 🍔 **เบอร์เกอร์** | ความขมตัดกับความมันของเนื้อ |
| 🌮 **ทาโก้** | ฮอปซิตรัส เข้ากับซัลซ่าและมะนาว |
| 🧀 **เชดดาร์แก่** | ฮอปสู้กับความเข้มของชีส |
| 🐖 **BBQ ซี่โครงหมู** | หวานรมควัน + ขม = มหัศจรรย์ |
| 🍟 **ของทอด** | คาร์บอนและความขมตัดน้ำมัน |
| 🌶️ **อาหารรสจัด** | ความขมช่วยบาลานซ์ความเผ็ด |

## 💡 เคล็ดลับการดื่ม

- **เสิร์ฟเย็นกำลังดี** — 8–12°C เย็นไปจะเสียกลิ่นฮอป
- **ยิ่งสดยิ่งอร่อย** — กลิ่นฮอปจางลง ดื่มภายใน 2–3 เดือนหลังบรรจุ
- **กินกับอาหารรสจัด** — เบียร์นี้อยู่กับของแซ่บได้สบาย
- **เหมาะกับคนรักฮอป** — ถ้าชอบความขม นี่คือสไตล์ของคุณ
- **ลองเทียบกับ Hazy IPA** — จะเห็นสเปกตรัมของ IPA
