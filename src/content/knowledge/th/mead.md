# 🍯 Mead

<div class="article-lead">

Mead คือ **เครื่องดื่มหมักที่เก่าแก่ที่สุดในประวัติศาสตร์มนุษยชาติ** — เก่าแก่กว่าเบียร์และไวน์หลายพันปี ผลิตจากการหมักน้ำผึ้งกับน้ำ ไม่ใช่เบียร์และไม่ใช่ไวน์ แต่เป็นอะไรที่แตกต่างไปโดยสิ้นเชิง: **ไวน์น้ำผึ้ง** ที่มีตั้งแต่หวานจัด ดื่มง่าย ไปจนถึงแห้ง ซ่า หรือแม้กระทั่งใส่ฮอป

หัวใจของ Mead เรียบง่าย: น้ำผึ้ง + น้ำ + ยีสต์ = มหัศจรรย์ แต่ในความเรียบง่ายนี้กลับมีความหลากหลายไม่รู้จบ

</div>

---

## 🏯 ประวัติโดยย่อ

Mead ถูกดื่มในแทบทุกอารยธรรมโบราณ:

- **จีนโบราณ** — ภาชนะดินเผาอายุ 7,000 ปีก่อนคริสตกาลพบร่องรอยการหมักน้ำผึ้ง
- **ไวกิ้ง** — Mead คือเครื่องดื่มของนักรบและเทพเจ้า เกี่ยวข้องกับบทกวีและปัญญา
- **กรีกโบราณ** — เรียกว่า *ambrosia* เครื่องดื่มของเทพเจ้า
- **ยุโรปยุคกลาง** — เชื่อกันว่า Mead ให้สุขภาพ ความอุดมสมบูรณ์ และพลัง

คำว่า **"ฮันนีมูน"** มาจากประเพณีนอร์สที่คู่บ่าวสาวจะดื่ม Mead เป็นเวลาหนึ่งเดือนหลังแต่งงาน เพื่อความอุดมสมบูรณ์และโชคดี

หลังจากเสื่อมความนิยมหลายศตวรรษ ปัจจุบัน Mead กำลัง **กลับมาได้รับความนิยมอีกครั้ง** ในยุคคราฟต์ มีการทดลองใช้ผลไม้ เครื่องเทศ ฮอป และการบ่มในถัง

## 👅 รสชาติเป็นอย่างไร?

รสชาติของ Mead แตกต่างกันอย่างมาก ขึ้นอยู่กับแหล่งที่มาของน้ำผึ้งและส่วนผสมที่เพิ่มเติม:

<div class="taste-box">

### 🎨 ลักษณะ
- สีฟางอ่อนถึงอำพันเข้มหรือทอง
- ใสหรือขุ่นก็ได้ (ถ้าไม่กรอง)
- นิ่งหรือซ่าได้

### 👃 กลิ่น
- **น้ำผึ้ง** — ดอกไม้ หวาน บางครั้งสมุนไพร
- **ขึ้นอยู่กับสิ่งที่เพิ่ม** ผลไม้ (เบอร์รี่ ซิตรัส พีช) เครื่องเทศ (อบเชย กานพลู วานิลลา) ดอกไม้ (ลาเวนเดอร์ กุหลาบ) หรือฮอป
- การหมักสะอาด ไม่มีกลิ่นแปลก

### 👄 รสชาติ
- **รสน้ำผึ้ง** — แตกต่างตามแหล่งดอกไม้ (wildflower, orange blossom, clover ฯลฯ)
- **หวานถึงแห้ง** — ขึ้นอยู่กับน้ำตาลตกค้าง
- **ขมน้อยมากถึงไม่มี** — เว้นแต่ใส่ฮอป
- **รสผลไม้ เครื่องเทศ หรือสมุนไพร** — ถ้ามีการเพิ่ม
- จบท้ายสะอาด คริสป์ หรือเข้มข้น ตามสไตล์

### ✨ ความรู้สึก
- **บอดี้เบาถึงเต็ม** — แตกต่างกันมาก
- นิ่ง ซ่าเล็กน้อย หรือซ่าเต็มที่
- บางชนิดมีแทนนินเล็กน้อย (จากผลไม้หรือการบ่ม)

</div>

## 🍶 ประเภทของ Mead

| ประเภท | คำอธิบาย |
|---|---|
| **Traditional Mead** | น้ำผึ้ง น้ำ ยีสต์ — รสน้ำผึ้งแท้ ๆ |
| **Melomel** | Mead ผสมผลไม้ (เบอร์รี่ เชอร์รี่ ซิตรัส ฯลฯ) |
| **Metheglin** | Mead ผสมเครื่องเทศหรือสมุนไพร (อบเชย ขิง วานิลลา) |
| **Bochet** | Mead จากน้ำผึ้งคาราเมล — รสลึก หอมคั่ว |
| **Cyser** | Mead ผสมแอปเปิล (คล้ายน้ำผึ้ง + ไซเดอร์) |
| **Pyment** | Mead ผสมองุ่น (คล้ายน้ำผึ้ง + ไวน์) |
| **Hydromel** | Mead แอลกอฮอล์ต่ำ (3.5–7.5% ABV) — เบา สดชื่น |
| **Sack Mead** | Mead แอลกอฮอล์สูง (14–18% ABV) — เข้มข้น |
| **Braggot** | Mead ผสมมอลต์ — ลูกผสมระหว่าง Mead และเบียร์ |
| **Hopped Mead** | Mead ที่ผ่านการ dry-hop เหมือน IPA |

## 🍯 น้ำผึ้ง: หัวใจของ Mead

ชนิดของน้ำผึ้งกำหนดรสชาติมากกว่าปัจจัยอื่น:

| แหล่งน้ำผึ้ง | รสชาติ |
|---|---|
| **Orange Blossom** | เบา หอมซิตรัส ดอกไม้ |
| **Wildflower** | ซับซ้อน เอิร์ธโทน ดอกไม้ |
| **Clover** | อ่อน หวาน สะอาด |
| **Buckwheat** | เข้ม เหมือนกากน้ำตาล เอิร์ธโทน |
| **Manuka** | สมุนไพร เข้มข้น |
| **Acacia** | เบามาก ละเอียดอ่อน ดอกไม้ |

## 🍽️ อาหารที่เข้ากัน

Mead เข้ากับอาหารได้หลากหลาย ขึ้นอยู่กับสไตล์

| ประเภท Mead | อาหารที่เข้ากัน |
|---|---|
| **Traditional แห้ง** | ชาร์คูเตอรี ชีสอ่อน ไก่ย่าง |
| **Traditional หวาน** | บลูชีส ทาร์ตผลไม้ ครีมบรูว์เล |
| **Melomel (เบอร์รี่)** | ดาร์กช็อกโกแลต เนื้อย่าง ขนมเบอร์รี่ |
| **Melomel (ซิตรัส)** | ซีฟู้ด สลัด อาหารรสจัด |
| **Metheglin (เครื่องเทศ)** | หมูอบ เครื่องเทศ เค้ก |
| **Hopped Mead** | เบอร์เกอร์ ของทอด อาหารเผ็ด |
| **Braggot** | BBQ ชีสแก่ สตูว์ |

## 💡 เคล็ดลับการดื่ม

- **อุณหภูมิสำคัญ** — Mead เบา: 8–12°C สดชื่น Mead เข้มข้น: 12–16°C ให้กลิ่นหอม
- **ให้มันหายใจ** — โดยเฉพาะ Mead ที่บ่มนานหรือแอลกอฮอล์สูง สัก 2–3 นาทีในแก้วสร้างความแตกต่าง
- **ลองกับอาหาร** — Mead เข้ากับอาหารได้หลากหลาย
- **อย่าคาดหวังว่าเป็นเบียร์หรือไวน์** — Mead มีเอกลักษณ์ของตัวเอง
- **Mead ซ่า** — ทางเลือกที่ดีแทนแชมเปญหรือสปาร์กลิงไวน์
