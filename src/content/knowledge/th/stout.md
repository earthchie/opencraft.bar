# ☕ Stout

<div class="article-lead">

Stout คือเบียร์ที่ **ดูน่ากลัวแต่ดื่มกลับนุ่ม surprising** ดำทะมึนราวกับอัศวินกลางคืน แต่ซ่อนรสชาติของกาแฟคั่ว ช็อกโกแลต และคาราเมล ไว้ในเนื้อเบียร์ที่เนียนนุ่ม ดื่มง่ายอย่างไม่น่าเชื่อ

ถ้าไม่เคยลองเบียร์ดำ Stout คือจุดเริ่มต้นที่ดีที่สุด

</div>

---

## 🏯 ประวัติโดยย่อ

Stout กำเนิดที่ **ไอร์แลนด์และอังกฤษ** ในศตวรรษที่ 18 วิวัฒนาการมาจาก Porter — เบียร์ดำที่ได้รับความนิยมในหมู่ชนชั้นแรงงานลอนดอน — แต่มี **แอลกอฮอล์สูงกว่าและข้าวบาร์เลย์คั่วมากกว่า** ทำให้มีรสชาติที่เข้มข้นและโดดเด่นกว่า

ชื่อ "Stout" เดิมหมายถึง "เข้มแข็ง" หรือ "ภูมิฐาน" — "Stout Porter" คือ Porter ที่เข้มข้นกว่าเดิม เมื่อเวลาผ่านไปคำว่า "Porter" ก็ถูกละทิ้ง และ **Stout กลายเป็นสไตล์ของตัวเอง**

Stout ที่โด่งดังที่สุดในโลก? **Guinness** ผลิตครั้งแรกที่ดับลินในปี 1759 ปัจจุบันโรงเบียร์คราฟต์ทั่วโลกนำสไตล์นี้ไปสร้างสรรค์ต่อ ตั้งแต่ Milk Stout หวานนุ่ม ไปจนถึง Imperial Stout บ่มในถัง

---

## 👅 รสชาติเป็นอย่างไร?

<div class="taste-box">

### 🎨 ลักษณะ
- **ดำสนิท** — ทึบแสงเหมือนเอสเพรสโซ
- ฟองครีมหนาสีแทนถึงน้ำตาล
- อาจมีประกายทับทิมเมื่อส่องแสง

### 👃 กลิ่น
- **มอลต์คั่ว** — กาแฟ ดาร์กช็อกโกแลต ขนมปังปิ้ง
- **คาราเมล** — หวานหอม
- **โอ๊ตหรือครีม** — ถ้ามีในสูตร
- กลิ่นฮอปน้อย — เอิร์ธโทนหรือสมุนไพร
- **กลิ่นแอลกอฮอล์อุ่น ๆ** — ในรุ่น ABV สูง

### 👄 รสชาติ
- **คั่ว** — เหมือนกาแฟดำดี ๆ สักแก้ว
- **ช็อกโกแลต** — เข้ม ขมหวาน หรือนม ขึ้นอยู่กับสไตล์
- **คาราเมล** — ความหวานเล็กน้อยบาลานซ์ความคั่ว
- **ความขมน้อยถึงปานกลาง** — จากข้าวบาร์เลย์คั่ว ไม่ใช่ฮอป
- **จบท้ายเนียน** — แห้ง (Dry Stout) หรือหวานเล็กน้อย (Sweet Stout)

### ✨ ความรู้สึก
- **บอดี้เบาถึงเต็ม** — Dry Stout เบา surprisingly ส่วน Imperial Stout หนาเนียน
- **ครีมมี่ เนียน** — โดยเฉพาะกับไนโตรเจน (Nitro)
- คาร์บอนต่ำถึงปานกลาง

</div>

## 🍶 ประเภทของ Stout

| ประเภท | คำอธิบาย |
|---|---|
| **Dry Stout / Irish Stout** | บอดี้เบา จบท้ายแห้ง ขมเหมือนกาแฟ คลาสสิกอย่าง Guinness |
| **Sweet Stout / Milk Stout** | ครีมมี่ หวานเล็กน้อย แลคโตสเพิ่มความนุ่ม |
| **Oatmeal Stout** | เนียน เต็ม หวานเล็กน้อย โอ๊ตเพิ่มความ silky |
| **Chocolate Stout** | รสช็อกโกแลตเข้ม — จากช็อกโกแลตมอลต์หรือช็อกโกแลตจริง |
| **Coffee Stout** | ใส่กาแฟหรือ cold brew — คั่วเข้มจัด |
| **Imperial Stout** | ABV สูง (8–12%+) เข้มข้น เต็ม-bodied มักบ่มถัง |
| **Oyster Stout** | ใส่หอยนางรม — ความเค็ม минералเข้ากับความคั่ว |
| **Pastry Stout** | แนวของหวาน — วานิลลา มะพร้าว เมเปิ้ล เนยถั่ว ฯลฯ |

## 🍶 Stout vs Porter

| ด้าน | Stout | Porter |
|---|---|---|
| สี | ดำสนิท | น้ำตาลเข้มถึงดำ |
| บอดี้ | เบาถึงเต็ม | ปานกลางถึงเต็ม |
| ความคั่ว | สูง จัดจ้าน | ปานกลาง เนียน |
| แอลกอฮอล์ | 4–12%+ | 4–8% |
| ความขม | ปานกลาง | ต่ำถึงปานกลาง |
| รสชาติเด่น | กาแฟ ช็อกโกแลต โทสต์ | ช็อกโกแลต ถั่ว คาราเมล |

เส้นแบ่งระหว่างสองสไตล์นี้บางมาก — ผู้ผลิตบางรายเรียกเบียร์ว่า "Stout" เพียงเพราะมันดำหรือเข้มกว่า

## 🍽️ อาหารที่เข้ากัน

Stout เป็นหนึ่งในสไตล์เบียร์ที่เข้ากับอาหารได้ดีที่สุด โดยเฉพาะอาหารรสเข้มและหวาน

| เมนู | ทำไมถึงเข้ากัน |
|---|---|
| 🦪 **หอยนางรม** | คู่คลาสสิก — เค็ม + คั่ว = มหัศจรรย์ |
| 🍫 **ดาร์กช็อกโกแลต** | กาแฟและช็อกโกแลตในเบียร์สะท้อนของหวาน |
| 🍨 **ไอศกรีม** | โดยเฉพาะ Milk Stout หรือ Chocolate Stout |
| 🧀 **ชีสแก่** | บลูชีสหรือเชดดาร์แก่สู้ความคั่วได้ดี |
| 🥩 **สเต็กย่าง** | ความคั่วเข้ากับกลิ่นย่าง smoky |
| 🍪 **บราวนี / คุกกี้** | ช็อกโกแลต + stout = สวรรค์ |

## 💡 เคล็ดลับการดื่ม

- **เสิร์ฟเย็นกำลังดี** — 10–13°C เย็นไปจะเสียความซับซ้อนของรสคั่ว
- **Nitro เปลี่ยนเกม** — การใช้ไนโตรเจนคาร์บอเนชันสร้างเนื้อครีมลื่นไหล หา Nitro Stout ลอง!
- **เหมาะกับการเก็บนาน** — Imperial Stout พัฒนาได้ดีข้ามปี ซื้อเพิ่มแล้วเก็บไว้
- **อย่ากลัวเบียร์ดำ** — Stout หลายตัวบอดี้เบากว่าที่คิด
- **คู่กับของหวาน** — Stout คือสไตล์เบียร์ที่ดีที่สุดสำหรับของหวาน
